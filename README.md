## Running

### Clone
```
git clone https://github.com/rTiRe/tickets_task.git
cd tickets_task
```
### Prepare for work

For Linux:
```
python3 -m venv venv
. ./venv/bin/activate
python -m pip install -r requirements.txt
```

For Windows:
```
python -m pip install virtualenv
python -m virtualenv venv
venv/Scripts/activate
python -m pip install -r requirements.txt
```

### Run script
```
python main.py
```
